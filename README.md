# DockOpsWebsite 🚀

DockOpsWebsite is a beginner-friendly DevOps project that demonstrates:

- Linux project setup
- Git & GitHub version control
- Docker containerization
- Docker Compose usage
- Basic CI/CD workflow understanding

---

## 📌 Project Overview

This project shows how a static HTML website can be:

1. Created in Linux
2. Managed using Git
3. Uploaded to GitHub
4. Containerized using Docker
5. Deployed using Docker Compose

---

## 📂 Project Structure

```
DockOpsWebsite/
│── Dockerfile
│── docker-compose.yml
│── README.md
└── html/
    └── index.html
```

---

## 🐳 How to Run the Project

Clone the repository:

```bash
git clone https://github.com/kayalasurekha/DockOpsWebsite.git
cd DockOpsWebsite
```

Build and start containers:

```bash
docker-compose up --build -d
```

Open your browser and visit:

```
http://localhost:8080
```
